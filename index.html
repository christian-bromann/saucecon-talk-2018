  • So you might be wondering hey, why is Selenium not on this chart?
  • The reason for that is that Seleniums is not mainly test framework
  • It is a utility tool that allows you to send command to a WebDriver endpoint to automate the browser
  • Which is why is a dependency of populare test frameworks like Protractor or Nightwatch
Selenium is software that is an implementation of the WebDriver protocol for automating browsers. One use of browser automation is testing websites, but there is nothing "official" about ​it doing​ that in any form​. As a result, I'd like to see Selenium do a better job of distinguishing itself as "browser automation" instead of "test automation."
